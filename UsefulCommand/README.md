### \# S3 버켓 오너십 변경
- A 계정에서 B 계정 소유의 버켓에 파일 업로시에 B 계정이 소유하게 변경

```
import boto3

client = boto3.client('s3')

response = client.put_bucket_ownership_controls(
    Bucket= bucket,
    OwnershipControls={
        'Rules': [
            {
                'ObjectOwnership': 'BucketOwnerPreferred'
            },
        ]
    }
)
```
